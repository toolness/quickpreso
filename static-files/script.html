When Atul first started knitting, everything looked like gobbledygook. Over time he learned how threads wove into a fabric. A teacher showed him how to hack and debug his knits with a crochet hook if he missed a stitch. And he was able to start crafting his own things. Knitting is a bit like coding. Everything looks like an incoherent jumble at first. But soon you perceive the connections between things. And understand how individual parts come together to form a cool thing. But knitting is also a bit more tangible than coding. It's harder to trace a program's path through code than it is to trace a thread of yarn through a knit. We should build tools that make it easier to understand the connections between different parts of code, so that knitters can be webmakers too.
